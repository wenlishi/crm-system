## ⚡ 快速启动（3 步）

### 步骤 1：启动 CRM 系统

**方式 A：IDEA 启动（推荐）**
```
1. 打开 IDEA
2. File → Open → 选择 crm-system 文件夹
3. 等待 Maven 导入完成
4. 找到 src/main/java/com/crm/system/SystemApplication.java
5. 点击运行按钮
6. 等待看到 "Started SystemApplication" 日志
```

**方式 B：Maven 启动**
```bash
cd /home/ubuntu/.openclaw/workspace/crm-system
mvn spring-boot:run
```

**方式 C：Docker 启动**
```bash
cd /home/ubuntu/.openclaw/workspace/crm-system
docker-compose up -d
```

## 🎯 如果遇到问题

### 问题 1：端口被占用

**错误**：`Port 8080 was already in use`

**解决**：
```bash
# 查找占用端口的进程
lsof -i :8080

# 杀死进程
kill -9 <PID>
```

---

### 问题 2：数据库连接失败

**错误**：`Communications link failure`

**解决**：
```bash
# 启动 MySQL
# Docker 方式
docker-compose up -d mysql

# 或修改配置文件使用本地 MySQL
```

---

### 问题 3：脚本执行失败

**错误**：`Permission denied`

**解决**：
```bash
chmod +x api-demo-script.sh
./api-demo-script.sh
```
