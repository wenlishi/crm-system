## 📊 前端集成示例（ECharts）

### 1. 销售排行榜（柱状图）

```javascript
// 获取销售排行榜数据
axios.get('/api/statistics/sales-ranking', {
  headers: { Authorization: getToken() }
}).then(res => {
  const data = res.data.data
  
  // 渲染柱状图
  const chart = echarts.init(document.getElementById('ranking-chart'))
  chart.setOption({
    xAxis: {
      type: 'category',
      data: data.map(item => item.username)
    },
    yAxis: {
      type: 'value',
      name: '成交金额'
    },
    series: [{
      data: data.map(item => item.totalAmount),
      type: 'bar',
      label: { show: true }
    }]
  })
})
```

### 2. 销售漏斗（漏斗图）

```javascript
// 获取转化率数据
axios.get('/api/statistics/conversion-rate').then(res => {
  const stages = res.data.data.funnelStages
  
  const chart = echarts.init(document.getElementById('funnel-chart'))
  chart.setOption({
    series: [{
      type: 'funnel',
      data: stages.map(stage => ({
        name: stage.stageName,
        value: stage.count
      }))
    }]
  })
})
```

### 3. 客户来源（饼图）

```javascript
// 获取来源数据
axios.get('/api/statistics/customer-source').then(res => {
  const sources = res.data.data.sources
  
  const chart = echarts.init(document.getElementById('source-chart'))
  chart.setOption({
    series: [{
      type: 'pie',
      data: sources.map(source => ({
        name: source.source,
        value: source.count
      }))
    }]
  })
})
```

### 4. 业绩趋势（折线图）

```javascript
// 获取趋势数据
axios.get('/api/statistics/performance-trend?type=day').then(res => {
  const trend = res.data.data
  
  const chart = echarts.init(document.getElementById('trend-chart'))
  chart.setOption({
    xAxis: {
      type: 'category',
      data: trend.map(item => item.date)
    },
    yAxis: { type: 'value' },
    series: [
      {
        name: '新增客户',
        type: 'line',
        data: trend.map(item => item.newCustomers)
      },
      {
        name: '成交金额',
        type: 'line',
        data: trend.map(item => item.amount)
      }
    ]
  })
})
```

## 🎯 关键指标说明

### 1. 销售排行榜

| 指标 | 说明 |
|------|------|
| totalCustomers | 客户总数 |
| dealCustomers | 成交客户数 |
| dealRate | 成交率 = 成交客户/总客户 |
| totalContracts | 合同总数 |
| totalAmount | 合同总金额 |
| rank | 排名 |

### 2. 转化率

| 指标 | 计算公式 |
|------|---------|
| 客户→商机转化率 | 商机数/客户数 × 100% |
| 商机→合同转化率 | 合同数/商机数 × 100% |
| 总体转化率 | 成交合同数/客户数 × 100% |

### 3. 客户来源

| 指标 | 说明 |
|------|------|
| count | 该来源的客户数 |
| percentage | 占比 = 该来源客户数/总客户数 |
| dealCount | 成交数量 |
| dealRate | 成交率 = 成交数/该来源客户数 |
