## 📊 API 接口（7 个）

| 接口 | 方法 | 权限 | 说明 |
|------|------|------|------|
| `/api/depts/tree` | GET | `system:dept:query` | 查询部门树 |
| `/api/depts/list` | GET | `system:dept:query` | 查询部门列表 |
| `/api/depts/{id}` | GET | `system:dept:query` | 查询部门详情 |
| `/api/depts` | POST | `system:dept:add` | 新增部门 |
| `/api/depts` | PUT | `system:dept:edit` | 更新部门 |
| `/api/depts/{id}` | DELETE | `system:dept:delete` | 删除部门 |
| `/api/depts/stats` | GET | `system:dept:query` | 部门统计 |

## 🎯 使用示例

### 1. 新增部门

```bash
curl -X POST http://localhost:8080/api/depts \
  -H "Content-Type: application/json" \
  -H "Authorization: $TOKEN" \
  -d '{
    "deptName": "广州分公司",
    "deptCode": "gz",
    "parentId": 1,
    "sortOrder": 4,
    "status": 1
  }'
```

### 2. 更新部门

```bash
curl -X PUT http://localhost:8080/api/depts \
  -H "Content-Type: application/json" \
  -H "Authorization: $TOKEN" \
  -d '{
    "deptId": 5,
    "deptName": "销售一部",
    "deptCode": "sales1",
    "parentId": 2,
    "leaderId": 100,
    "leaderName": "张三",
    "phone": "13800138000"
  }'
```

### 3. 删除部门

```bash
curl -X DELETE http://localhost:8080/api/depts/5 \
  -H "Authorization: $TOKEN"
```

**注意**：如果有子部门，需要先删除子部门

### 4. 查询部门统计

```bash
curl http://localhost:8080/api/depts/stats \
  -H "Authorization: $TOKEN"
```

**响应**：
```json
{
  "code": 200,
  "data": {
    "total": 8,
    "active": 7,
    "disabled": 1
  }
}
```

## 🎯 最佳实践

### 1. 部门编码规范

```
总公司：head
分公司：地区缩写（bj/sh/sz/gz）
部门：英文缩写（sales/tech/market/service）
```

### 2. 部门层级建议

```
建议不超过 4 级：
- 第 1 级：总公司/集团
- 第 2 级：分公司/事业部
- 第 3 级：部门
- 第 4 级：小组/团队
```

### 3. 数据权限集成


## 🔧 前端集成示例（Vue3）

### 部门树组件

```vue
<template>
  <el-tree
    :data="deptTree"
    :props="{ label: 'deptName', children: 'children' }"
    node-key="deptId"
    @node-click="handleNodeClick"
  />
</template>

<script setup>
import { ref, onMounted } from 'vue'
import axios from 'axios'

const deptTree = ref([])

onMounted(async () => {
  const res = await axios.get('/api/depts/tree', {
    headers: { Authorization: getToken() }
  })
  deptTree.value = res.data.data
})

const handleNodeClick = (data) => {
  console.log('选中部门：', data.deptName)
}
</script>
```
