### 7. 全局异常处理增强

**建议**：创建错误码枚举

```java
public enum ErrorCode {
    SUCCESS(200, "成功"),
    
    // 认证错误 (401-499)
    UNAUTHORIZED(401, "未授权"),
    TOKEN_EXPIRED(401, "Token 已过期"),
    
    // 权限错误 (403-499)
    FORBIDDEN(403, "权限不足"),
    
    // 资源错误 (404-499)
    NOT_FOUND(404, "资源不存在"),
    CUSTOMER_NOT_FOUND(404, "客户不存在"),
    
    // 业务错误 (500-599)
    INTERNAL_ERROR(500, "服务器错误"),
    DATA_EXISTS(500, "数据已存在");
    
    private final int code;
    private final String message;
}
```
