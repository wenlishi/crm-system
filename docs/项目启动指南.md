## 🔧 安装步骤

### 1. 安装 JDK 17

```bash
sudo apt update
sudo apt install -y openjdk-17-jdk

# 验证
java -version
# 应该显示：openjdk version "17.x.x"
```

### 2. 安装 MySQL

```bash
sudo apt install -y mysql-server

# 启动 MySQL
sudo systemctl start mysql
sudo systemctl enable mysql

# 验证
mysql --version
# 应该显示：mysql  Ver 8.0.x
```

### 3. 安装 Redis

```bash
sudo apt install -y redis-server

# 启动 Redis
sudo systemctl start redis
sudo systemctl enable redis

# 验证
redis-cli ping
# 应该返回：PONG
```

### 4. 安装 Maven（如果需要）

```bash
sudo apt install -y maven

# 验证
mvn -version
```

## 🐛 常见问题

### 1. 端口被占用

**错误信息**：`Port 8080 was already in use`

**解决方案**：
```bash
# 查找占用端口的进程
lsof -i :8080

# 杀死进程
kill -9 <PID>

# 或者修改端口
# 在 application.yml 中修改 server.port: 8081
```

### 2. MySQL 连接失败

**错误信息**：`Communications link failure`

**解决方案**：
```bash
# 检查 MySQL 是否启动
sudo systemctl status mysql

# 启动 MySQL
sudo systemctl start mysql

# 检查用户名密码
# 在 application.yml 中确认 username 和 password 正确
```

### 3. Redis 连接失败

**错误信息**：`Cannot get Jedis connection`

**解决方案**：
```bash
# 检查 Redis 是否启动
redis-cli ping

# 启动 Redis
sudo systemctl start redis
```

### 4. Maven 依赖下载慢

**解决方案**：使用阿里云镜像

编辑 `~/.m2/settings.xml`：
```xml
<mirrors>
  <mirror>
    <id>aliyun</id>
    <name>Aliyun Maven</name>
    <url>https://maven.aliyun.com/repository/public</url>
    <mirrorOf>central</mirrorOf>
  </mirror>
</mirrors>
```
