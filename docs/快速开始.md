# 🚀 CRM 项目快速开始指南

**更新时间**：2026-03-14

---

## 📋 第一步：克隆项目

```bash
# 进入工作目录
cd /home/ubuntu/.openclaw/workspace

# 克隆项目（如果你已经创建了 GitHub 仓库）
git clone https://github.com/YOUR_USERNAME/crm-system.git

# 或者直接使用本地项目
cd crm-system
```

---

## 📦 第二步：检查开发环境

### 必需软件

**1. JDK 17+**
```bash
java -version
# 应该显示：java version "17.x.x"
```

如果没有安装：
```bash
# Ubuntu/Debian
sudo apt update
sudo apt install openjdk-17-jdk

# macOS
brew install openjdk@17
```

**2. Maven 3.8+**
```bash
mvn -version
# 应该显示：Apache Maven 3.8.x
```

如果没有安装：
```bash
# Ubuntu/Debian
sudo apt install maven

# macOS
brew install maven
```

**3. MySQL 8.0+**
```bash
mysql --version
# 应该显示：mysql  Ver 8.0.x
```

**4. Redis 6.0+**
```bash
redis-server --version
# 应该显示：Redis server v=6.x.x
```

## 🗄️ 第三步：初始化数据库

```bash
# 1. 登录 MySQL
mysql -u root -p

# 2. 创建数据库并导入表结构
source /home/ubuntu/.openclaw/workspace/crm-system/docs/sql/schema.sql

# 3. 验证
USE crm_system;
SHOW TABLES;
# 应该看到：sys_user, sys_role, crm_customer, crm_follow_up, crm_opportunity, crm_contract 等表
```

## ⚙️ 第四步：配置项目

### 创建 application.yml

```bash
mkdir -p src/main/resources
```

创建文件 `src/main/resources/application.yml`：

```yaml
spring:
  application:
    name: crm-system
  
  # 数据源配置
  datasource:
    url: jdbc:mysql://localhost:3306/crm_system?useSSL=false&serverTimezone=Asia/Shanghai&characterEncoding=utf8&allowPublicKeyRetrieval=true
    username: root
    password: your_password  # ⚠️ 修改为你的 MySQL 密码
    driver-class-name: com.mysql.cj.jdbc.Driver
  
  # Redis 配置
  redis:
    host: localhost
    port: 6379
    password: 
    database: 0
    timeout: 10s
    lettuce:
      pool:
        max-active: 8
        max-idle: 8
        min-idle: 0
  
  # JPA 配置
  jpa:
    hibernate:
      ddl-auto: none
    show-sql: true
    properties:
      hibernate:
        format_sql: true
  
  # Jackson 配置
  jackson:
    date-format: yyyy-MM-dd HH:mm:ss
    time-zone: Asia/Shanghai

# MyBatis-Plus 配置
mybatis-plus:
  mapper-locations: classpath*:/mapper/**/*.xml
  type-aliases-package: com.crm.system.entity
  global-config:
    db-config:
      id-type: assign_id
      logic-delete-field: deleted
      logic-delete-value: 1
      logic-not-delete-value: 0
  configuration:
    map-underscore-to-camel-case: true
    log-impl: org.apache.ibatis.logging.stdout.StdOutImpl

# 服务器配置
server:
  port: 8080
  servlet:
    context-path: /api

# 日志配置
logging:
  level:
    root: INFO
    com.crm.system: DEBUG
  pattern:
    console: '%d{yyyy-MM-dd HH:mm:ss} [%thread] %-5level %logger{50} - %msg%n'
```

## 🚀 第六步：创建启动类

创建文件 `src/main/java/com/crm/system/SystemApplication.java`：

```java
package com.crm.system;

import org.mybatis.spring.annotation.MapperScan;
import org.springframework.boot.SpringApplication;
import org.springframework.boot.autoconfigure.SpringBootApplication;

@SpringBootApplication
@MapperScan("com.crm.system.mapper")
public class SystemApplication {

    public static void main(String[] args) {
        SpringApplication.run(SystemApplication.class, args);
        System.out.println("====================================");
        System.out.println("✅ CRM 系统启动成功！");
        System.out.println("📍 访问地址：http://localhost:8080/api");
        System.out.println("====================================");
    }
}
```

## 🐛 常见问题

### 1. 端口被占用
```bash
# 查看占用 8080 端口的进程
lsof -i :8080

# 杀死进程
kill -9 <PID>

# 或者修改端口
# 在 application.yml 中修改 server.port
```

### 2. MySQL 连接失败
```bash
# 检查 MySQL 是否启动
sudo systemctl status mysql

# 启动 MySQL
sudo systemctl start mysql

# 检查用户名密码是否正确
# 在 application.yml 中修改 spring.datasource.username 和 password
```

### 3. Redis 连接失败
```bash
# 检查 Redis 是否启动
redis-cli ping
# 应该返回：PONG

# 启动 Redis
redis-server

# 或者修改配置
# 在 application.yml 中修改 spring.redis.host 和 port
```

### 4. Maven 依赖下载慢
```bash
# 使用阿里云镜像
# 在~/.m2/settings.xml 中添加：
<mirrors>
  <mirror>
    <id>aliyun</id>
    <name>Aliyun Maven</name>
    <url>https://maven.aliyun.com/repository/public</url>
    <mirrorOf>central</mirrorOf>
  </mirror>
</mirrors>
```
