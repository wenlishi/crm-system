## 🚀 如何使用

### 在 IDEA 中运行（推荐）

1. **打开项目**
   ```
   File → Open → 选择 crm-system 文件夹
   ```

2. **等待 Maven 导入完成**

3. **运行应用**
   ```
   找到 SystemApplication.java
   点击运行按钮
   ```

4. **访问 Swagger**
   ```
   http://localhost:8080/api/swagger-ui.html
   ```

5. **运行测试**
   ```
   右键测试文件 → Run 'xxxTest'
   ```

### 运行集成测试

```bash
# 运行认证模块测试
mvn test -Dtest=AuthControllerIntegrationTest

# 运行客户模块测试
mvn test -Dtest=CustomerControllerIntegrationTest

# 运行所有测试
mvn test
```
