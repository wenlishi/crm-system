## ✅ 已完成功能模块

### 1. 认证模块（3 个接口）

**功能**：用户登录、注册、信息查询

| 接口 | 方法 | 说明 |
|------|------|------|
| `/auth/login` | POST | 用户登录 |
| `/auth/register` | POST | 用户注册 |
| `/auth/info` | GET | 获取当前用户信息 |

**技术亮点**：
- ✅ JWT Token 认证
- ✅ BCrypt 密码加密
- ✅ Token 自动刷新

### 2. 客户管理模块（6 个接口）

### 5. 合同管理模块（8 个接口）

## 🚀 如何使用

### 1. 环境准备

```bash
# JDK 17+
java -version

# Maven 3.8+
mvn -version

# MySQL 8.0+
mysql --version

# Redis 6.0+
redis-server --version
```

### 2. 安装步骤

```bash
# 克隆项目
git clone https://github.com/wenlishi/crm-system.git
cd crm-system

# 创建数据库
mysql -u root -p < docs/sql/schema.sql

# 修改配置
vim src/main/resources/application.yml

# 启动项目
mvn spring-boot:run

# 测试接口
./test-api.sh
./test-contract.sh
```

### 3. 访问地址

- **后端接口**：http://localhost:8080/api
- **默认管理员**：admin / admin123
